Sistemas Personalizados no Mini Kraken Bot para RPG de Mesa no discord.

Introdução

Poucas pessoas sabem, porém é possível ter seu próprio sistema personalizado com o Mini Kraken, este aqui é um guia pratico de como criar e adicionar o seu sistema Homebrew ao bot, e ter todos os recursos de fichas inteligentes sobe medida para sua mesa, de maneira facil, rápida e intuitiva.

Alerta! Papo para Nerds

Os sistemas do Mini Kraken usam arquivos JSON para criar e manipular o funcionamento de um sistema, portanto para criar um novo sistemas é necessário escrever um JSON do mesmo e o guia a seguir vai explicar um pouco do seu funcionamento e estrutura para permitir que crie seus próprios sistemas de RPG nele.

i18n
